EPUB 3 Validator

Validates a EPUB.

Synopsis

Inputs

  • EPUB
    EPUB to be validated.
    Media-type:
    • application/epub+zip
    • application/oebps-package+xml
    Sequence: no

Outputs

  • HTML Report
    An HTML-formatted version of the validation report.
    Media-type:
    • application/vnd.pipeline.report+xml
  • Validation status
    Validation status (http://code.google.com/p/daisy-pipeline/wiki/ValidationStatusXML).
    Media-type:
    • application/vnd.pipeline.status+xml
