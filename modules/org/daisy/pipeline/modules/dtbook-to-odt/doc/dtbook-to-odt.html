DTBook to ODT

Transforms a DTBook (DAISY 3 XML) document into an ODT (Open Document Text).

Synopsis

Inputs

  • source
    Input DTBook.
    Media-type:
    • application/x-dtbook+xml
    Sequence: no

Outputs

  • output-dir
    Directory for storing result files.
    Media-type:

Options

  • template
    OpenOffice template file (.ott) that contains the style definitions.
    Data-type: <string>
    Default value:
    Required: no
  • asciimath
    How to render ASCIIMath-encoded formulas?
    Data-type: One of the following:
    ASCIIMATH
    MATHML
    Default value: ASCIIMATH
    Required: no
  • images
    How to render images?
    Data-type: One of the following:
    EMBED
    LINK
    Default value: EMBED
    Required: no
  • page-numbers
    Show page numbers or not.
    Data-type: <boolean>
    Default value: true
    Required: no
  • page-numbers-float
    Try to float page numbers to an appropriate place as opposed to exactly following print.
    Data-type: <boolean>
    Default value: true
    Required: no
  • image-dpi
    Resolution of images in DPI.
    Data-type: <integer>
    Default value: 600
    Required: no
