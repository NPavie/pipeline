Style sheets can be associated with an HTML file in several ways: linked (using an 'xml-stylesheet' processing instruction or a 'link' element), embedded (using a 'style' element) and/or inlined (using 'style' attributes). Document-specific CSS takes precedence over any CSS provided through the "Style sheets" option.
