Transforms ZedAI XML (ANSI/NISO Z39.98-2012 Authoring and Interchange) into HTML.

Synopsis

Inputs

  • ZedAI document
    Input ZedAI.
    Media-type:
    • application/z3998-auth+xml
    Sequence: no

Outputs

  • HTML
    The resulting HTML file.
